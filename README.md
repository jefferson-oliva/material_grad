# material_grad

Neste repositório estão disponíveis materiais de aulas ministradas na Universidade Tecnológica Federal do Paraná (UTFPR), campus de Pato Branco.

Esses materiais são referentes às seguintes displinas do curso de Engenharia de Computação:

- Algoritmos e Estrutura de Dados I (AE22CP)
- Algoritmos e Estrutura de Dados II (AE23CP)

Obs.: várias notas de aula devem ser atualizadas, já que parte do conteúdo estão nos slides.


A seguir, são apresentados links de listas de exercícios do HackerRank:

- AE22CP: https://www.hackerrank.com/ae22cp-utfpr-pb

- AE23CP: https://www.hackerrank.com/ae23cp-utfpr-pb
